<Section icon={Target} title="Parcours de progression pour débutants">

Se lancer dans un nouveau jeu de combat 2D peut être difficile. Suivez ce parcours de progression structuré pour développer vos compétences avant de tenter votre chance dans les matchs classés en ligne [1.1].

<Steps>
  <Step num="1" title="Choisissez votre personnage principal">
    Choisissez un combattant simple comme Aang ou Zuko pour apprendre les mouvements de base, les attaques normales et les coups spéciaux simples [1.1].
  </Step>
  <Step num="2" title="Équipez un personnage de soutien">
    Associez votre combattant à un personnage de soutien qui couvre vos faiblesses. Utilisez-les pour prolonger les combos, échapper à la pression ou contrôler l'espace [1.1].
  </Step>
  <Step num="3" title="Maîtrisez la mécanique de Flow">
    Entraînez-vous à utiliser le Flow pour sprinter, esquiver les projectiles et lancer des attaques aériennes. Un bon mouvement est la base du jeu compétitif [1.1].
  </Step>
  <Step num="4" title="Terminez les modes Histoire et Arcade">
    Jouez à travers les campagnes solo. Le mode Histoire propose un récit écrit par Michael Dante DiMartino et Tim Hedrick, tandis que le mode Arcade offre des fins individuelles pour chaque personnage [1.1].
  </Step>
  <Step num="5" title="Analysez les données en mode Entraînement">
    Prenez vos affrontements préférés dans le mode Entraînement pour étudier les hitboxes, les données de frame et le timing précis des combos [1.1].
  </Step>
</Steps>

<Callout type="success" title="Jeu local Versus">

</Callout>
