<Section icon={Target} title="Guide des mécaniques de combat">

Le succès dans Avatar Legends: The Fighting Game repose sur la maîtrise du système de Flow et la gestion de votre jauge d'Énergie. Comprendre ces mécaniques de base est la première étape vers le jeu compétitif.

<Steps>
  <Step num="1" title="Maîtriser le mouvement Flow">
    Utilisez le système de Flow pour dasher, esquiver les projectiles et vous repositionner. C'est la base de votre jeu défensif et offensif.
  </Step>
  <Step num="2" title="Gérer les points d'Énergie">
    Accumulez de l'Énergie grâce aux coups réussis et aux blocages. Dépensez-la judicieusement pour des Coups Spéciaux Améliorés ou conservez-la pour un Art Super capable de renverser le match.
  </Step>
  <Step num="3" title="S'entraîner en mode Entraînement">
    Utilisez les données de frame, la visualisation des hitboxes et les états de sauvegarde pour affiner vos combos et apprendre les interactions spécifiques à chaque match.
  </Step>
</Steps>

| Mécanique | Objectif | Conseil pour débutants |
| :--- | :--- | :--- |
| **Flow** | Évasion & Mouvement | Utiliser pour échapper à la pression |
| **Énergie** | Amélioration de capacité | Économiser pour les gros combos |
| **Arts Super** | Finition cinématique | Utiliser pour punir les erreurs |
| **Soutiens** | Aide tactique | Associer pour couvrir les faiblesses |

<Callout type="warning" title="Gestion de l'Énergie">
Ne gaspillez pas votre jauge d'Énergie dans des attaques aléatoires. Le jeu de haut niveau récompense les joueurs qui attendent le moment optimal pour utiliser leurs Arts Super.
</Callout>
