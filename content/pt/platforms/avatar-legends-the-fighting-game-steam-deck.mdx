<Section icon={Target} title="Configurações Otimizadas para Modo Portátil">

Obter a melhor experiência em **avatar legends the fighting game steam deck** requer alguns ajustes nas configurações do jogo e do SteamOS. Como o jogo utiliza recursos 2D desenhados à mão, a carga sobre a GPU é relativamente leve, permitindo que você aumente outras configurações ou economize bateria.

<Steps>
  <Step num="1" title="Resolução e Redimensionamento">
    Defina a resolução do jogo para **1280x800**. Certifique-se de que o Filtro de Redimensionamento no Menu de Acesso Rápido (QAM) do Steam Deck esteja definido como 'Linear' ou 'Integer' para obter os sprites 2D mais nítidos.
  </Step>
  <Step num="2" title="Limite de Taxa de Quadros">
    Defina a Taxa de Atualização do Steam Deck para **60Hz** e o Limite de Quadros para **60**. Nunca jogue um jogo de luta a 30 FPS ou com taxa de quadros destravada, pois isso quebra o tempo do motor de combate.
  </Step>
  <Step num="3" title="V-Sync e Input Lag">
    Desative o V-Sync no menu do jogo se estiver usando o limitador de quadros integrado do Steam Deck. Isso reduz a latência de entrada, o que é vital para reagir a ataques aéreos (overheads) e projéteis.
  </Step>
  <Step num="4" title="Predefinições Gráficas">
    Defina a 'Qualidade dos Efeitos' para Médio e o 'Detalhe do Cenário' para Alto. Os personagens são a prioridade; manter os elementos de fundo ligeiramente mais baixos ajuda a garantir 60 FPS cravados durante os Super Arts mais intensos.
  </Step>
</Steps>

<Callout type="tip" title="Otimização de Bateria">
Se estiver jogando offline no Modo História, você pode reduzir o TDP (Thermal Design Power) para **8W ou 10W** para estender a vida útil da bateria sem sacrificar a meta de 60 FPS.
</Callout>

</Section>

<Section icon={Swords} title="Mecânicas de Combate e Mapeamento de Controles">

O combate em Avatar Legends é construído em torno da **Mecânica de Fluxo (Flow Mechanic)**, que permite movimentos fluidos, esquivas rápidas e sequências de golpes específicas de cada personagem. No Steam Deck, o D-pad é geralmente preferido para comandos direcionais precisos, enquanto os quatro botões frontais cuidam dos seus ataques principais.

Cada personagem também pode escolher um **Personagem de Suporte** (como Appa Samurai ou Sokka) para fornecer assistências táticas. Mapear essas assistências para os botões traseiros do Steam Deck (L4/R4) pode proporcionar uma vantagem significativa, permitindo que você ative os suportes sem tirar os polegares dos botões de ataque principais.

| Ação | Mapeamento Padrão | Sugestão para Steam Deck |
| :--- | :--- | :--- |
| **Ataque Leve** | X / Quadrado | Botão X |
| **Ataque Forte** | Y / Triângulo | Botão Y |
| **Especial/Dobra** | B / Círculo | Botão B |
| **Corrida de Fluxo** | RB / R1 | R1 ou R4 (Traseiro Direito) |
| **Assistência de Suporte** | LB / L1 | L1 ou L4 (Traseiro Esquerdo) |
| **Super Art** | Combo de Gatilhos | L2 + R2 |

<Callout type="info" title="Pontos de Energia">
