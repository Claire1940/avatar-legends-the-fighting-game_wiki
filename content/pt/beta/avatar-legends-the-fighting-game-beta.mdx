<Section icon={Sparkles} title="Mecânicas de Combate Principais: Flow e Energia">

O combate na **beta de avatar legends the fighting game** é construído sobre dois pilares principais: o sistema de Flow (Fluxo) e os Pontos de Energia. Ao contrário dos jogos de luta tradicionais que dependem exclusivamente de comandos com precisão de frames, este título enfatiza a movimentação e o gerenciamento tático de recursos.

<CardGrid cols={2}>
  <Card title="O Sistema de Flow">
    - **Movimentação**: Permite investidas instantâneas e desvios por baixo de projéteis.
    - **Cancelamentos**: O Flow pode ser usado para cancelar os frames de recuperação de movimentos especiais.
    - **Específico de Personagem**: Cada dobrador tem opções exclusivas de Flow (ex: Patinetes de Ar para Aang).
  </Card>
  <Card title="Pontos de Energia (EP)">
    - **Acúmulo**: Obtido ao acertar ataques e realizar bloqueios com sucesso.
    - **Especiais Aprimorados**: Gaste 1 barra de EP para adicionar propriedades como armadura ou acertos extras.
    - **Super Artes**: Gaste a barra cheia para um finalizador cinematográfico de alto dano.
  </Card>
</CardGrid>

<Callout type="success" title="Mecânica Avançada: Cancelamentos de Dobra">
Jogadores experientes podem usar o Flow para cancelar um projétil de dobra em uma investida física. Esta é a principal maneira de "reiniciar" sua pressão e manter os oponentes adivinhando no canto da tela.
</Callout>

Cada estilo de dobra também dita como o personagem interage com o cenário. Dobradores de Terra podem criar pilares temporários que agem como barreiras físicas, enquanto Dobradores de Água podem manipular a trajetória de seus projéteis no meio do voo. Essas nuances estão totalmente disponíveis para testes nos modos Treino e Versus da beta.

</Section>

<Section icon={Shield} title="Requisitos de Sistema e Recursos Online">

Para participar da **beta de avatar legends the fighting game** no PC, seu sistema deve atender a metas de hardware específicas. O jogo utiliza animações 2D feitas à mão, o que exige menos das placas de vídeo do que títulos em 3D, mas requer um desempenho estável do processador para manter o bloqueio de 60 FPS necessário para o funcionamento do rollback netcode.

| Componente | Requisitos Mínimos | Especificações Recomendadas |
| :--- | :--- | :--- |
| **SO** | Windows 10 (64-bit) | Windows 11 (64-bit) |
| **Processador** | Intel Core i5-4460 | Intel Core i7-8700K |
| **Memória** | 8 GB de RAM | 16 GB de RAM |
| **Placa de Vídeo** | NVIDIA GeForce GTX 760 | NVIDIA GeForce RTX 2060 |
| **Armazenamento** | 20 GB de espaço disponível | 20 GB (SSD Recomendado) |

O jogo online é a peça central da beta. Os desenvolvedores estão usando este período para ajustar os seguintes recursos:

- **Rollback Netcode**: Tecnologia proprietária projetada para minimizar o atraso de comandos, mesmo em conexões de alta latência.
- **Cross-play**: Partidas simplificadas entre usuários de Steam, PlayStation e Xbox.
- **Modo Espectador**: Permite que os jogadores assistam a partidas de alto nível em tempo real.
- **Sistema de Replay**: Salve e analise suas partidas para avaliar dados de frames e erros.

<Callout type="info" title="Requisitos de Console">
Observe que os jogadores de PS5 precisam de uma assinatura ativa do PlayStation Plus para jogar online, e os jogadores de Xbox precisam do Xbox Game Pass (Core ou Ultimate).
</Callout>
