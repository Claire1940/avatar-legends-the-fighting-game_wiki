<Section icon={Target} title="Mecânicas de Combate Principais: Dominando o Flow">

O sucesso em **Avatar Legends the Fighting Game Online** depende de dominar a interação entre o sistema de Flow e o gerenciamento de Energia. Ao contrário dos jogos de luta tradicionais que dependem muito de comandos direcionais complexos, este jogo usa um esquema de controle simplificado que enfatiza tempo, posicionamento e gasto de recursos.

<Steps>
  <Step num="1" title="Entenda o Sistema de Flow">
    O Flow é a mecânica de movimentação universal. Ao pressionar um único botão dedicado, os jogadores podem dar investidas, esquivar-se por baixo de projéteis ou transicionar para sequências específicas de cada personagem. Dominar o Flow é essencial para passar por personagens focados em controle de espaço (zoning), como Katara.
  </Step>
  <Step num="2" title="Gerencie os Pontos de Energia (EP)">
    O EP é acumulado ao acertar ataques e realizar bloqueios com sucesso. Você pode gastar 1 barra para usar **Enhanced Special Moves** (adicionando armadura ou dano) ou guardar barras cheias para uma **Super Art**, um finalizador cinematográfico que pode mudar o rumo do round.
  </Step>
  <Step num="3" title="Utilize Personagens de Suporte">
    Antes de uma partida, você seleciona um Personagem de Suporte (como Appa ou Momo). Eles não são jogáveis, mas oferecem chamados de assistência que fornecem escudos defensivos, pokes ofensivos ou bônus de movimentação.
  </Step>
  <Step num="4" title="Domine os Detalhes de Cada Dobra">
    Cada elemento tem uma propriedade única. Os movimentos da Dobra de Terra frequentemente possuem "Super Armor", enquanto os movimentos da Dobra de Ar permitem pulos duplos e flutuação. Aprenda essas propriedades para explorar as fraquezas elementais do seu oponente.
  </Step>
</Steps>

| Mecânica | Propósito | Dica para Iniciantes | Aplicação Avançada |
| :--- | :--- | :--- | :--- |
| **Flow Dash** | Movimentação Evasiva | Use para desviar de projéteis | Cancele ataques normais em Flow para combos |
| **Enhanced Specials** | Aumento de Poder | Use para finalizar um combo | Use para obter vantagem de frames no bloqueio |
| **Super Arts** | Dano Cinemático | Use apenas quando for nocautear | Use para punir movimentos com longa recuperação |
| **Support Assist** | Ajuda Tática | Use quando estiver encurralado | Use para estender combos de malabarismo (juggle) |

<Callout type="success" title="Profundidade Mecânica">
O sistema de "Flow" permite **Flow Cancels**, que deixam os jogadores de alto nível interromperem uma animação de ataque pela metade para enganar o oponente ou estender uma sequência de combo. Esta é a chave para alcançar o ranque Diamante na liga online.
</Callout>

</Section>

<Section icon={Shield} title="Infraestrutura Online: Rollback & Crossplay">

Para qualquer jogo de luta moderno, a experiência online é o componente mais crítico. **Avatar Legends the Fighting Game Online** utiliza uma solução proprietária de rollback netcode, que prevê os comandos dos jogadores para eliminar aquela sensação de lentidão dos sistemas baseados em atraso (delay-based). Isso garante que até mesmo partidas entre jogadores de regiões diferentes permaneçam responsivas.

O jogo também conta com **Crossplay** completo, o que significa que os jogadores de Steam, PlayStation e Xbox compartilham a mesma fila de matchmaking. Isso mantém os tempos de espera curtos e a comunidade competitiva unificada.

| Recurso | Disponibilidade | Detalhes |
| :--- | :--- | :--- |
| **Rollback Netcode** | Sim | Sistema proprietário para latência mínima |
| **Crossplay entre Plataformas** | Sim | Fila unificada para PC, PS5 e Xbox |
| **Matchmaking Ranqueado** | Sim | Divisões baseadas em habilidade com recompensas sazonais |
| **Modo Espectador** | Sim | Assista a partidas ao vivo de alto nível dentro do jogo |
| **Teatro de Replays** | Sim | Salve e analise suas lutas anteriores |

<Callout type="info" title="Requisitos de Console">
Observe que a assinatura do **PlayStation Plus** é necessária para jogar online no PS5, e o **Xbox Game Pass** (Core ou superior) é necessário para jogar online nos consoles Xbox. Jogadores do Steam não precisam de assinatura.
</Callout>

</Section>
