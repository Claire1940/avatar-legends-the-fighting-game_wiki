<Section icon={Target} title="Guía de mecánicas de combate">

El éxito en Avatar Legends: The Fighting Game depende de dominar el sistema de Fluidez (Flow) y gestionar tu medidor de Energía. Comprender estas mecánicas principales es el primer paso hacia el juego competitivo.

<Steps>
  <Step num="1" title="Domina el movimiento de Fluidez (Flow)">
    Usa el sistema de Fluidez para arremeter (dash), esquivar proyectiles y reposicionarte. Esta es la base de tu juego ofensivo y defensivo.
  </Step>
  <Step num="2" title="Gestiona los puntos de Energía">
    Acumula Energía mediante golpes y bloqueos exitosos. Gástala sabiamente en movimientos especiales potenciados o ahórrala para un Arte Súper (Super Art) que cambie el rumbo del combate.
  </Step>
  <Step num="3" title="Practica en el modo de entrenamiento">
    Utiliza los datos de fotogramas (frame data), la visualización de cajas de golpe (hitboxes) y los estados de guardado para perfeccionar tus combos y aprender interacciones específicas de cada enfrentamiento.
  </Step>
</Steps>

| Mecánica | Propósito | Consejo para principiantes |
| :--- | :--- | :--- |
| **Fluidez (Flow)** | Evasión y movimiento | Úsala para escapar de la presión |
| **Energía** | Mejora de habilidades | Guárdala para combos grandes |
| **Artes Súper (Super Arts)** | Remate cinematográfico | Úsalas para castigar errores |
| **Apoyos** | Asistencia táctica | Emparéjalos para cubrir debilidades |

<Callout type="warning" title="Gestión de energía">
No desperdicies tu medidor de Energía en ataques aleatorios. El juego de alto nivel recompensa a los jugadores que esperan el momento óptimo para usar sus Artes Súper.
</Callout>

</Section>
