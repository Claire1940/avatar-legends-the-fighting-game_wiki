<Section icon={Target} title="Optimal Settings for Handheld Play">

Getting the best experience on **avatar legends the fighting game steam deck** requires a few tweaks to the in-game and SteamOS settings. Because the game uses hand-drawn 2D assets, the GPU load is relatively light, allowing you to push other settings or save battery.

<Steps>
  <Step num="1" title="Resolution and Scaling">
    Set the in-game resolution to **1280x800**. Ensure that the Scaling Filter in the Steam Deck Quick Access Menu (QAM) is set to 'Linear' or 'Integer' for the sharpest 2D sprites.
  </Step>
  <Step num="2" title="Frame Rate Limit">
    Set the Steam Deck's Refresh Rate to **60Hz** and the Frame Limit to **60**. Never play a fighting game at 30 FPS or with an unlocked frame rate, as it breaks the combat engine's timing.
  </Step>
  <Step num="3" title="V-Sync and Input Lag">
    Disable V-Sync in the game menu if you are using the Steam Deck's built-in frame limiter. This reduces input latency, which is vital for reacting to overheads and projectiles.
  </Step>
  <Step num="4" title="Graphic Presets">
    Set 'Effects Quality' to Medium and 'Background Detail' to High. The characters are the priority; keeping background assets slightly lower can help maintain a locked 60 FPS during intensive Super Arts.
  </Step>
</Steps>

<Callout type="tip" title="Battery Optimization">
If playing offline in Story Mode, you can lower the TDP (Thermal Design Power) to **8W or 10W** to extend battery life without sacrificing the 60 FPS target.
</Callout>

</Section>
